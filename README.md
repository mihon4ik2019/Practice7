1. **Клонируйте репозиторий**
   ```sh
   git clone <URL_REPOSITORY>
   cd <REPOSITORY_FOLDER>
   ```
2. **Установите http-server** (если не установлен):
   ```sh
   npm install -g http-server
   ```
3. **Запустите сервер**
   ```sh
   http-server -p 8080
   ```
5. **Откройте в браузере**
   http://localhost:8080
